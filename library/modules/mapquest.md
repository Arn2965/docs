# Mapquest

This module returns to you the Latitude and Longitude coordinates for an address / location.

## Settings

* **Mode**
  * Options - Geocoding

## Inputs

* **Address** - Accepts a location/address \(e.g. Brooklyn NY, 112 Happy Street Brooklyn NY, etc.\) 

## Outputs

* Latitude
* Longitude
