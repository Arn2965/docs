# Trello

## Link Your Trello Account

When you add a Trello Trigger to your program, you will be prompted to link an account if you have not done so already. Get My Boards

## Get My Boards

### Outputs

* Board Names
* Board IDs

## Get Cards From a Board

Select a Board

### Outputs

* Card Names
* Card IDs
* Card Descriptions
* Card Lists

## Create a New Card

1. Select a Board
2. Select a List

### Inputs

* Card Name
* Card Description
* Due Date for the Card
