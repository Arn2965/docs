## 游댍 Search for Videos by Keyword

### 游닌 Inputs

* Search Term
* Maximum Number of Results
* Advanced:
  * Sort Order
  * Filter Results by U.S. Zip Code

### 游닋 Outputs

* Video Title
* Author Channel Username
* Video URL
* Video Description
* Date Published
* Author Channel ID
* Video ID

### 游닌 Inputs
